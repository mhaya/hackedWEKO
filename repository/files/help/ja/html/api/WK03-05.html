概要

WEKOアイテム検索とはWEKOに登録されたアイテムのメタデータおよびファイルの内容から、指定したキーワードで検索を行う機能となります。また、タイトル、著者など特定の項目とキーワードを指定して検索することもできます。

OpenSearchとは検索結果を他のシステムで利用できるようにするための仕組みとなります。

本書はWEKOアイテム検索についてのOpensearchに関するAPI仕様を記載した文書となります。

 

 

クエリ仕様

WEKOアイテム検索のOpenSearchクエリは以下の形式です。

[トップページURL]?action=repository_opensearch&(パラメータ=値)&(パラメータ=値)&…&(パラメータ=値)

 

クエリを構成する各パラメータは以下のとおりです。記述例に関してはURLエンコードされる前の値を記載しております。

なお、デフォルト値が空となっているパラメータを指定しなかった場合、NULLとして扱われます。

また、一つのパラメータで複数の値を指定する場合、カンマ区切りで指定します。

    1         

list_view_num

ページあたりの検索結果件数を指定するパラメータ（任意）

 

20

list_view_num=50

指定可能な値は20、50、75、100である。

20以下の値が指定された場合は20として処理する。

20～50の間の値が指定された場合は50として処理し、50～75、75～100の間の値が指定された場合も同様に、75、100として処理する。

100以上の値が指定された場合は100として処理する。

指定されない場合はデフォルトで設定されている表示件数が表示される。
